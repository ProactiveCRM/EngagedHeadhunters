# Engaged Headhunters

This is a premium executive search and recruiting platform built with [Next.js](https://nextjs.org).

## Features
- **Premium Salary Guide**: Market-accurate compensation data for executive leadership.
- **Outcome-Focused Messaging**: Driven by the "Industrial Pulse" aesthetic.
- **Enterprise-Grade Infrastructure**: Integrated with Supabase and Next.js App Router.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Methodology
Our compensation data is based on real placements and market intelligence, ensuring deep insights into executive salary trends.
